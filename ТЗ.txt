	Приложение должно осуществлять поиск книг по базе данных по автору, названию и по году написания. Слева должны быть кнопки, возвращающие предыдущие параметры фильтра. В результате поиска должен выводиться результат в виде таблицы со столбцами аналогичного вида (автор, название, год). Ячейки таблицы должны быть неизменяемыми, но иметь функционал нажатия по ним: нажатие на ячейку с именем автора или с датой написания запускает поиск по базе данных с этим фильтром, нажатие на название должно открывать книгу.
	Меню должно состоять из трёх кнопок: файл, позволяющий открывать и сохранять книги или библиотеки целиком, главная, позволяющая изменять формат текста (размер, шрифт и цвет) и кнопка помощь, содержащая один пункт - справка.
	Открытие файла происходит так: появляется всплывающее окно с вариантами поиска - библиотека (.lib) или книга (.txt, .docx, (+ может быть, другие расширения)), затем появляется окно выбора файлов, причём, при фыборе неверного формата файла пользователь увидит сообщение об ошибке. Удаление книг осуществляется выбором книг галочками слева и нажатием кнопки "Удалить", появляющейся внизу.
	Смена формата текста и справка открываются в виде всплывающих окон.


	Текст книги открывается в отдельном окне. Здесь только одна кнопка меню - формат текста (аналогична кнопке главная из основного окна) и поле для вывода текста (+ кнопки внизу для перелистывания страниц).
