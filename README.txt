Приложение библиотека. Позволяет читать и добавлять книги.
Есть база данных из пяти таблиц:
1. Основная (ID - Автор - Название - Год написания - Файл с текстом книги - Файл с картинкой обложки);
2. Список авторов (ID - Автор);
3. Список дат (ID - Год написания);
4. Список файлов с текстом (ID - Название файла);
4. Список файлов с картинками (ID - Название файла).

Файл библиотеки имеет расширение ".lib". Это zip-файл, содержащий базу данных библиотеки и две папки (папка с текстами книг и папка с картинками обложек)

Поиск книг осушествляется по фильтрам "автор", "название" и "год".
После фильтрации по базе данных пользователю будет доступен список подходящих по фильтрам книг.
Обложка с текстом будут доступны для чтения во всплывающем окне.

Добавление книг будет осуществляться через вкладку "файл" в меню. Программа спросит у пользователя автора, название, год написания, а затем два раза откроется окно выбора файлов для поиска текста и обложки.
